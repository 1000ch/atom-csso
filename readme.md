## Install

```bash
$ apm install csso
```

## Usage

- Packages > CSSO > **Minify CSS**
- Context Menu > **Minify CSS**
- Command Palette (<kbd>⌘</kbd> + <kbd>shift</kbd> + <kbd>P</kbd>) > **Minify CSS**
